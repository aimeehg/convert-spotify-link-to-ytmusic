# Telegram bot to convert a Spotify link to YTMusic link
This Telegram bot gets a Spotify song link and replies with a YouTube Music link

It's a simple bot. I'm going to add more features in the future.

[Telegram Bot](https://t.me/convertmusiclink_bot)

## Usage
    - Send a message to the bot with a Spotify link like this: https://open.spotify.com/intl-es/track/0l7HoOCQOKZpg5MV2hXw57?si=4b70ddf9c47545a2
    - The bot replies with the YTMusic link!
